📝 วิธีส่งผลหวย (ใช้ทุกวัน)
  1. กดแท็บ "กรอกผล" ที่แถบด้านล่าง
  2. เปิด 🎨 Style Studio เพื่อตกแต่งสไตล์ เลือกธีมสี 5 แบบ (🧁 Macaroon / 🍬 Candy / 🌊 Ocean / ✨ Gold / 🌙 Dark) เลือกฟอนต์ 3 แบบ (● มน / ◆ คม / ○ เส้น) + ขนาด (S / M / L) กด 🎲 สุ่มสไตล์ เพื่อให้ระบบเลือกแบบสุ่มให้ ดู Preview สด — เห็นผลทันทีตอนเปลี่ยนค่า
  3. หาหวยที่ต้องการ (พิมพ์ชื่อในช่องค้นหา) → กรอก เลขบน / เลขล่าง ดู Preview ใต้ช่องกรอก — เห็นได้เลยว่า LINE กลุ่มจะเห็นแบบไหน
  4. กดปุ่ม "ส่งผล"
  5. ระบบส่งอัตโนมัติไปทั้ง Telegram และ LINE กลุ่ม LINE จะได้รับรูปตัวเลขตามสไตล์ที่เลือก + ข้อความ
  6. เช็คสถานะได้ที่หน้า "หน้าหลัก" (TG ✓ / LINE ✓)
