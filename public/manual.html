🤖 ระบบอัตโนมัติ (ไม่ต้องทำอะไร)
  1. หวยหุ้น 22 รายการ (นิเคอิ, ฮั่งเส็ง, ดาวโจนส์ ฯลฯ) ดึงราคาดัชนีหุ้นจริง → คำนวณเลข 3 ตัวบน + 2 ตัวล่าง อัตโนมัติ
  2. หวย Hanoi/Laos 21 รายการ (ฮานอย VIP, ลาว HD ฯลฯ) ดึงผลจากเว็บต้นทางอัตโนมัติด้วย Puppeteer
  3. ระบบทำงานผ่าน Cron ทุก 1 นาที ตามเวลาออกผลแต่ละหวย
  4. ผลถูกส่งไปทั้ง Telegram + LINE พร้อมรูปตัวเลขอัตโนมัติ
  5. เช็คสถานะได้ที่หน้า "หน้าหลัก" (ดูว่าหวยไหนส่งแล้ว / รอ)
🎨 ตั้งค่าธีมรูปตัวเลข
  1. กดแท็บ "ตั้งค่า" ที่แถบด้านล่าง
  2. เลื่อนลงไปที่ "🎨 สไตล์รูปตัวเลข (ผล auto)" เลือกธีมสี 6 แบบ (🎀 Shopee / 🧁 Macaroon / 🍬 Candy / 🌊 Ocean / ✨ Gold / 🌙 Dark) เลือกเรียงตัวเลข: แนวนอน หรือ แนวตั้ง เลือกฟอนต์ 3 แบบ (● มน / ◆ คม / ○ เส้น) + ขนาด (S / M / L)
  3. ดู Preview ด้านล่าง — เห็นตัวอย่างรูปที่จะส่งไป LINE ทันที
  4. ระบบจำค่าที่ตั้งไว้ → ใช้กับ ผลหวยทุกตัวที่ดึงอัตโนมัติ

💬 เพิ่มกลุ่ม LINE
  1. เปิดกลุ่ม LINE ที่ต้องการรับผลหวย
  2. เชิญ bot "หน้าบ้าน" เข้ากลุ่ม เพิ่มเพื่อน → เชิญเข้ากลุ่ม
  3. ระบบจับกลุ่มอัตโนมัติ — ไม่ต้องตั้งค่าอะไรเพิ่ม
เพิ่มได้หลายกลุ่ม ส่งผลพร้อมกันทุกกลุ่ม ถ้า bot ถูกเชิญออก เชิญเข้าใหม่ได้เลย
📅 ตั้งเวลาส่งข้อความ
  1. กดแท็บ "ตั้งเวลาส่ง" ที่แถบด้านล่าง
  2. เลือกหวยที่ต้องการตั้งเวลา → กำหนดวัน/เวลาที่ต้องการส่ง ระบบจะส่งข้อความไป Telegram + LINE อัตโนมัติตามเวลาที่ตั้ง
  3. ดูรายการที่ตั้งเวลาไว้ → แก้ไข / ลบได้ตลอด
ใช้สำหรับตั้งเวลาส่งข้อความล่วงหน้า เช่น ประกาศพิเศษ หรือแจ้งเตือนก่อนปิดรับ
⏰ Countdown แจ้งเตือนก่อนปิดรับ
  1. ระบบส่ง Countdown อัตโนมัติ 3 ครั้ง ก่อนเวลาปิดรับของแต่ละหวย 20 นาที / 10 นาที / 5 นาที ก่อนปิด
  2. ข้อความจะถูกส่งไปทั้ง Telegram + LINE กลุ่ม อัตโนมัติ
  3. ตั้งค่าเปิด/ปิด Countdown ได้ที่หน้า "ตั้งค่า" เปิด/ปิดรายหวย หรือปิดทั้งระบบได้
