🤖 ระบบอัตโนมัติ (ไม่ต้องทำอะไร)
  1. หวยหุ้น 22 รายการ (นิเคอิ, ฮั่งเส็ง, ดาวโจนส์ ฯลฯ) ดึงราคาดัชนีหุ้นจริง → คำนวณเลข 3 ตัวบน + 2 ตัวล่าง อัตโนมัติ
  2. หวย Hanoi/Laos 21 รายการ (ฮานอย VIP, ลาว HD ฯลฯ) ดึงผลจากเว็บต้นทางอัตโนมัติด้วย Browser (Puppeteer)
  3. ระบบทำงานผ่าน Cron ทุก 1 นาที ตามเวลาออกผลแต่ละหวย
  4. ผลถูกส่งไปทั้ง Telegram + LINE พร้อมรูปตัวเลข + emoji numbers อัตโนมัติ
  5. Countdown แจ้งเตือนก่อนปิดรับ — ตั้งค่าช่วงเวลาได้จากหน้าตั้งค่า ค่า default: 5 นาที — แก้ได้ที่ ตั้งค่า → "Countdown แจ้งเตือนก่อนปิดรับ" (ใส่เลขคั่นคอมมา เช่น 20,10,5)
  6. ตั้งเวลาส่งข้อความ — กำหนดเวลา + วัน ส่งข้อความอัตโนมัติซ้ำได้ เข้าที่หน้า "ตั้งเวลาส่ง" (/scheduled)
  7. เช็คสถานะได้ที่หน้า "หน้าหลัก" (ดูว่าหวยไหนส่งแล้ว / รอ)
  8. ส่งข้อความเอง / ประกาศพิเศษได้ที่หน้า "ข้อความ" พิมพ์ข้อความอะไรก็ได้ → เลือกส่ง LINE / Telegram / ทั้งสอง
  9. ตั้งค่าแยกแต่ละกลุ่ม LINE — เลือกหวยที่จะส่ง + ลิงก์เฉพาะกลุ่ม จัดการที่ ตั้งค่า → จัดการกลุ่ม (/groups)
🎨 ตั้งค่าธีมรูปตัวเลข
  1. กดแท็บ "ตั้งค่า" ที่แถบด้านล่าง
  2. เลื่อนลงไปที่ "🎨 สไตล์รูปตัวเลข (ผล auto)" เลือกธีมสี 8 แบบ (✍️ Outline / 🖤 Dark Minimal / 🎀 Shopee / 🧁 Macaroon / 🍬 Candy / 🌊 Ocean / ✨ Gold / 🌙 Dark) เลือกเรียงตัวเลข 3 แบบ: 🔺 Inline (แนะนำ) / ➡️ แนวนอน / ⬇️ แนวตั้ง เลือกฟอนต์ 21 แบบ (ฟอนต์ไทย: มะลิ, ไอติม, มิตร, คณิต ฯลฯ) หรือพิมพ์ชื่อ Google Font เอง ขนาดตัวเลข: S / M / L
  3. ดู Preview ด้านล่าง — เห็นตัวอย่างรูปที่จะส่งไป LINE ทันที
  4. ระบบจำค่าที่ตั้งไว้ → ใช้กับ ผลหวยทุกตัวที่ดึงอัตโนมัติ

💬 เพิ่มกลุ่ม LINE
  1. เปิดกลุ่ม LINE ที่ต้องการรับผลหวย
  2. เชิญ bot "หน้าบ้าน" เข้ากลุ่ม เพิ่มเพื่อน → เชิญเข้ากลุ่ม
  3. ระบบจับกลุ่มอัตโนมัติ — ไม่ต้องตั้งค่าอะไรเพิ่ม
  4. จัดการกลุ่มได้ที่ ตั้งค่า → จัดการกลุ่ม หรือหน้า /groups ตั้งค่าต่อกลุ่ม: ลิงก์กลุ่ม, เลือกหวยที่จะส่ง, เปิด/ปิดกลุ่ม
เพิ่มได้หลายกลุ่ม ส่งผลพร้อมกันทุกกลุ่ม ถ้า bot ถูกเชิญออก เชิญเข้าใหม่ได้เลย

📅 ตั้งเวลาส่งข้อความ
  1. เข้าหน้า "ตั้งเวลาส่ง" (จาก menu หรือ /scheduled) ตั้งเวลา + วัน → ระบบส่งข้อความอัตโนมัติตามกำหนด
  2. กำหนดข้อความ + เวลา + วันที่ต้องการส่ง ระบบจะส่งไป Telegram + LINE อัตโนมัติตามเวลาที่ตั้ง
  3. ดูรายการที่ตั้งเวลาไว้ → แก้ไข / ลบได้ตลอด
ใช้สำหรับตั้งเวลาส่งข้อความล่วงหน้า เช่น ประกาศพิเศษ หรือข้อความที่ต้องส่งซ้ำทุกวัน
⏰ Countdown แจ้งเตือนก่อนปิดรับ
  1. ระบบส่ง Countdown อัตโนมัติก่อนเวลาปิดรับของแต่ละหวย ค่า default: 5 นาที — ตั้งค่าได้ เช่น 20,10,5 (แจ้ง 3 ครั้ง) หรือ 5 (แจ้งครั้งเดียว)
  2. ข้อความจะถูกส่งไปทั้ง Telegram + LINE กลุ่ม อัตโนมัติ
  3. ตั้งค่าช่วงเวลาได้ที่ ตั้งค่า → "Countdown แจ้งเตือนก่อนปิดรับ" ใส่นาทีคั่นคอมมา เช่น 20,10,5 = แจ้ง 3 ครั้ง / ใส่ 5 = แจ้งครั้งเดียว

🔍 ตรวจสอบระบบ
  1. กดแท็บ "ตั้งค่า" ที่แถบด้านล่าง
  2. กดปุ่ม "🔍 ตรวจสอบระบบทั้งหมด" ระบบจะเช็คทุกอย่าง: DB, Telegram, LINE Endpoint, กลุ่ม, Scraping
  3. ดูผลลัพธ์ — ✅ เขียว = ปกติ / ⚠️ เหลือง = มีข้อสังเกต / ❌ แดง = ต้องแก้
แนะนำกดตรวจสอบเป็นประจำ เพื่อให้แน่ใจว่าระบบพร้อมส่งผลหวย
🔧 ระบบ Unofficial API (ส่งไม่จำกัด)
  1. ระบบส่งข้อความผ่าน Unofficial API — ไม่มี quota ส่งได้ไม่จำกัด ถ้า Unofficial ล้ม → ระบบสลับไปใช้ Official API อัตโนมัติ (Smart Fallback)
  2. Token refresh อัตโนมัติ — ไม่ต้องดูแล ระบบเช็คทุก 6 ชม. + refresh ก่อนหมดอายุ
  3. มีระบบ กันแบน — หน่วงเวลาสุ่มระหว่างกลุ่ม + แจ้งเตือนถ้ากลุ่มเกิน 15
  4. Heartbeat Monitor — เช็คสถานะทุก 5 นาที แจ้ง Telegram ถ้าระบบล่ม
