## 1. ภาพรวมระบบ

### LottoBot คืออะไร?
ระบบส่งผลหวยต่างประเทศอัตโนมัติเข้ากลุ่ม LINE ประกอบด้วย:

- 🤖 **Bot อัตโนมัติ** — ดึงผลหวย 43 รายการจากเว็บ/ตลาดหุ้น
- 📱 **ส่งเข้ากลุ่ม LINE** — ใช้บัญชี LINE ส่ง trigger → LINE OA ตอบผลหวยฟรี
- 📊 **Admin Dashboard** — กรอกผลมือ, ดูประวัติ, ตั้งค่า
- ⏰ **Countdown** — แจ้งเตือนก่อนหวยออก

### องค์ประกอบของระบบ

```
┌─────────────┐   ┌─────────────┐   ┌─────────────┐
│  Dashboard  │   │   VPS       │   │  LINE OA    │
│ (Vercel)    │◄──┤ (Singapore) ├──►│ (Webhook)   │
└─────────────┘   └─────────────┘   └─────────────┘
      │                 │                  │
      ▼                 ▼                  ▼
 Admin ใช้งาน     Bot ส่ง "."         Reply ผลหวย
                  เข้ากลุ่ม           เข้ากลุ่ม (ฟรี!)
```

## 2. เริ่มต้นใช้งาน

### 2.1 เข้า Dashboard
```
https://lottobot-chi.vercel.app/settings
```

### 2.2 Login ครั้งแรก (PIN Login)

1. **ปิด LINE Desktop** บนคอม (ถ้าเปิดอยู่ด้วยบัญชี bot)
2. เข้าหน้า **"ตั้งค่า"** → เลื่อนหา **"ตั้งค่าบัญชี LINE Bot"**
3. กรอก:
   - Email ที่ใช้สมัคร LINE
   - Password ที่ตั้งใน LINE
4. กด **🔑 PIN Login**
5. จะเห็น **PIN 6 หลัก** สีเหลือง
6. เปิด **LINE app บนมือถือ** → จะมี popup ถาม PIN → ใส่ PIN
7. รอ 2-3 วินาที → จะเห็น **"✅ Login สำเร็จ!"**

**เสร็จ!** ระบบพร้อมใช้งาน

### 2.3 เชิญ Bot เข้ากลุ่ม LINE

**2 บัญชีต้องอยู่ในกลุ่มเดียวกัน:**
1. บัญชีส่วนตัวที่ login (onepnk1) — สำหรับส่ง trigger
2. LINE Official Account (LottoBot) — สำหรับ reply ผลหวย

**วิธี:**
1. เปิดกลุ่ม LINE ที่ต้องการ
2. เมนู → เชิญ → เลือก:
   - บัญชี onepnk1 (บัญชีส่วนตัว)
   - @LottoBot (Official Account)
3. ระบบจะเห็นกลุ่มอัตโนมัติภายใน 1-2 นาที

### 2.4 ตรวจสอบว่าใช้งานได้

1. ใน Settings → เลื่อนหา **"วิธีส่งข้อความ LINE"**
2. ตรวจว่าเลือก **"🎯 Trigger (แนะนำ)"**
3. กดปุ่ม **🧪 ทดสอบ Trigger**
4. ถ้าเห็น **"Trigger สำเร็จ! ส่ง X/X กลุ่ม"** = ใช้งานได้ ✅

## 3. การใช้งานประจำวัน

### 3.1 ระบบทำงานอัตโนมัติ

**ไม่ต้องทำอะไรเลย** ระบบจะ:
- ดึงผลหวยทุกวันตามเวลาที่ตั้งไว้
- ส่งเข้ากลุ่ม LINE อัตโนมัติ
- บันทึกประวัติใน Dashboard

### 3.2 กรอกผลหวยเอง (Manual Entry)

บางหวย (ลาว, ฮานอย) ต้องกรอกผลเอง:
1. เข้าหน้า **"กรอกผล"**
2. เลือกหวย
3. กรอกเลข บน/ล่าง
4. กด **บันทึก + ส่ง**

ระบบจะส่งเข้ากลุ่มทันที

### 3.3 ดูประวัติการส่ง

หน้า **"ประวัติ"** แสดง:
- วันที่
- หวยที่ออก
- สถานะส่ง Telegram (Admin log)
- สถานะส่ง LINE (กลุ่ม)
- Error ถ้ามี

### 3.4 ตั้งค่ากลุ่ม LINE

หน้า **"ตั้งค่า"** → **"กลุ่ม LINE"**:
- เปิด/ปิด กลุ่มที่ใช้งาน
- ตั้งลิงค์แอดเพื่อน เฉพาะกลุ่ม
- กำหนดข้อความเพิ่มเติม

## 4. เมื่อเกิดปัญหา

### 🚨 ปัญหาที่ 1: Bot ไม่ส่งข้อความเข้ากลุ่ม

**อาการ:** หวยออกแล้ว แต่ไม่มีข้อความในกลุ่ม LINE

**สาเหตุที่เป็นไปได้:**
1. บัญชี Bot ถูก logout (session หลุด)
2. เปิด LINE Desktop พร้อมกับ bot (session ชน)
3. VPS ล่ม

**วิธีแก้ไขด่วน:**

#### Step 1: เช็คสถานะระบบ
เปิด: `http://45.77.240.100:8080/health`

ดู JSON:
- `"clientReady": true` ✅ = OK
- `"clientReady": false` ❌ = ต้อง re-login

#### Step 2: Re-login (ถ้า clientReady = false)

1. **ปิด LINE Desktop บนคอม** (ถ้าเปิดอยู่)
2. เข้า Settings: `https://lottobot-chi.vercel.app/settings`
3. เลื่อนหา **"ตั้งค่าบัญชี LINE Bot"**
4. กรอก email + password
5. กด **🔑 PIN Login**
6. Verify PIN ที่มือถือ
7. เสร็จ — ระบบกลับมาใช้งานได้

#### Step 3: ถ้ายังไม่ได้ — Restart VPS

1. เปิด Vultr Dashboard: https://my.vultr.com
2. เลือก server
3. กด **Restart** (รูป ⟳)
4. รอ 1 นาที
5. ทำ Re-login ซ้ำ (Step 2)

---

### 🚨 ปัญหาที่ 2: Bot ถูก LINE แบน

**อาการ:** กด PIN Login แล้วได้ error `ABUSE_BLOCK` หรือ `AUTHENTICATION_FAILED`

**สาเหตุ:** LINE detect ว่าเป็น bot → บล็อคชั่วคราว/ถาวร

**วิธีแก้:**

#### ทางแก้ 1: รอ 24 ชั่วโมง
LINE block บางครั้งเป็นแค่ชั่วคราว (1-24 ชม.) — ลอง login ใหม่พรุ่งนี้

#### ทางแก้ 2: สมัครบัญชี LINE ใหม่
ถ้าบล็อคถาวร — ต้องใช้บัญชีใหม่:
1. สมัคร LINE ด้วย**เบอร์ใหม่**
2. ตั้ง Email + Password ใน LINE settings
3. เชิญบัญชีใหม่เข้ากลุ่มที่ใช้งาน
4. Re-login ที่ Dashboard Settings ด้วยบัญชีใหม่

#### ทางแก้ 3: ลดโหลดการส่ง
ไปที่ Settings → **"วิธีส่งข้อความ LINE"**:
- ใช้ **Trigger mode** (ปลอดภัยสุด)
- ลดจำนวนกลุ่ม ≤ 10 กลุ่ม/บัญชี
- หลีกเลี่ยงการส่งเกิน 500 ครั้ง/วัน/บัญชี

---

### 🚨 ปัญหาที่ 3: VPS ล่ม (เว็บเข้าไม่ได้)

**อาการ:** `http://45.77.240.100:8080/health` → เข้าไม่ได้

**วิธีเช็ค:**
1. เข้า Vultr Dashboard
2. ดูสถานะ server:
   - **Running** (เขียว) = VPS ทำงาน → ปัญหาอยู่ที่ service
   - **Stopped** (แดง) = VPS หยุด → กด Start
   - **Installing** = รอ

**ถ้า VPS Running แต่เว็บเข้าไม่ได้:**
- Service (bot) อาจ crash
- ติดต่อ dev ให้ SSH เข้าไปรีสตาร์ท
- หรือกด **Restart** บน Vultr Dashboard → รอ 1 นาที → ลองใหม่

---

### 🚨 ปัญหาที่ 4: LINE OA ไม่ตอบ

**อาการ:** ส่ง "." เข้ากลุ่มได้ แต่ไม่มี reply จาก OA

**วิธีเช็ค:**
1. เข้า LINE app → กลุ่มที่ใช้งาน
2. ดูรายชื่อสมาชิก (เมนู ≡)
3. มี **@LottoBot** (LINE OA) ในกลุ่มไหม?

**ถ้าไม่มี:**
- เชิญ @LottoBot เข้ากลุ่ม
- Add friend @LottoBot ก่อน (ถ้ายังไม่ได้เพิ่ม)

**ถ้ามีแต่ไม่ตอบ:**
- Webhook อาจไม่ได้ตั้งค่า → ติดต่อ dev

## 6. คำถามที่พบบ่อย

### Q1: ใช้ LINE มือถือขณะ bot ทำงานได้ไหม?
**A:** ✅ ใช้ได้ปกติ บนมือถือเป็น Primary device ไม่ชนกับ bot (ที่เป็น Secondary)

### Q2: Bot แบนแล้วแก้ยังไง?
**A:** ดู [ปัญหาที่ 2](#-ปัญหาที่-2-bot-ถูก-line-แบน)

### Q3: รับได้กี่กลุ่ม?
**A:** แนะนำ **10-15 กลุ่ม/บัญชี bot** (ปลอดภัย)
ถ้าต้องการมากกว่านี้ → สมัครบัญชีเพิ่ม

### Q4: Token หมดอายุเมื่อไหร่?
**A:** Token มีอายุประมาณ 7 วัน → ระบบ auto-refresh เองถึง 1 ปี
หลังจากนั้นต้อง PIN Login ใหม่

### Q5: ลูกค้าออกจากไลน์ได้ไหม?
**A:** ✅ ออกจาก LINE มือถือได้ปกติ (กลับมา login ได้) แต่**ห้ามลบ account**

### Q6: ทำไมต้องปิด LINE Desktop?
**A:** LINE อนุญาต 1 Desktop session ต่อ account. Bot ใช้ slot นี้อยู่ ถ้าเปิด Desktop → bot โดน kick

### Q7: กลุ่มใหม่ต้องทำยังไง?
**A:** 
1. เชิญบัญชี bot + @LottoBot เข้ากลุ่ม
2. ระบบจะเห็นกลุ่มอัตโนมัติภายใน 1-2 นาที
3. เข้า Settings → เลื่อนหา "กลุ่ม LINE" → เช็คว่ากลุ่มใหม่อยู่ในรายการ
4. เปิด toggle (เขียว) = ใช้งาน

### Q8: ต้องการหยุด bot ชั่วคราว?
**A:**
- ปิดเฉพาะกลุ่ม: Settings → กลุ่ม LINE → toggle ปิด
- ปิดทั้งระบบ: Settings → ตั้งค่าหวย → ปิด scrape

### Q9: ข้อความเข้ากลุ่มช้า?
**A:** Trigger mode มีหน่วงเวลา 3-8 วินาที เพื่อป้องกันแบน เป็นเรื่องปกติ

### Q10: ค่าใช้จ่ายรายเดือน?
**A:**
- VPS Vultr: ~$6/เดือน (฿200)
- Vercel: ฟรี
- Supabase: ฟรี
- LINE OA: ฟรี (เพราะใช้ Reply API)
- **รวม: ~฿200/เดือน**
