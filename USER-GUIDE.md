## 1. วิธีเข้าใช้งาน Dashboard

1. เปิดเว็บเบราว์เซอร์ (แนะนำ Chrome หรือ Safari)
2. พิมพ์ URL ของ Dashboard ที่ได้รับ (ดูใน HANDOVER.md)
3. กรอก Email และ Password ที่ได้รับจากผู้พัฒนา
4. กด **เข้าสู่ระบบ**
5. จะเข้าสู่หน้าหลักของ Dashboard

> **หมายเหตุ:** หากลืมรหัสผ่าน ให้กด "ลืมรหัสผ่าน" ระบบจะส่งลิงก์รีเซ็ตไปยัง Email

---

## 2. วิธีดูสถานะหวยวันนี้

1. เข้า Dashboard
2. ที่หน้าแรก จะเห็น **สถานะหวยวันนี้** แสดงเป็นรายการ
3. แต่ละรายการจะแสดง:
   - **ชื่อหวย** — เช่น UK Lotto, Powerball
   - **เวลาออกผล** — เวลาที่หวยจะออก (ตามเวลาไทย)
   - **นับถอยหลัง** — เวลาที่เหลือก่อนออกผล
   - **สถานะ** — รอออกผล / ออกผลแล้ว / ส่งแล้ว / ส่งไม่สำเร็จ
4. หวยที่ออกผลแล้วจะแสดงตัวเลขผลลัพธ์ให้เห็นทันที

---

## 3. วิธีจัดการหวย (เพิ่ม / แก้ไข / ลบ / เปิด-ปิด)

### เพิ่มหวยใหม่

1. ไปที่เมนู **จัดการหวย**
2. กดปุ่ม **+ เพิ่มหวย**
3. กรอกข้อมูล:
   - ชื่อหวย
   - ประเทศ
   - URL เว็บต้นทาง (สำหรับ Scraping)
   - ตารางออกผล (วันและเวลา)
   - รูปแบบข้อความที่จะส่ง
4. กด **บันทึก**

### แก้ไขหวย

1. ไปที่เมนู **จัดการหวย**
2. หาหวยที่ต้องการแก้ไข
3. กดปุ่ม **แก้ไข** (ไอคอนดินสอ)
4. แก้ไขข้อมูลที่ต้องการ
5. กด **บันทึก**

### ลบหวย

1. ไปที่เมนู **จัดการหวย**
2. หาหวยที่ต้องการลบ
3. กดปุ่ม **ลบ** (ไอคอนถังขยะ)
4. ระบบจะถามยืนยัน — กด **ยืนยันลบ**

> **คำเตือน:** การลบหวยจะลบข้อมูลสถิติย้อนหลังทั้งหมดของหวยนั้นด้วย

### เปิด-ปิดหวย

1. ไปที่เมนู **จัดการหวย**
2. หาหวยที่ต้องการ
3. กดสวิตช์ **เปิด/ปิด** ที่ข้างชื่อหวย
   - **เปิด (สีเขียว)** = ระบบจะดึงผลและส่งตามปกติ
   - **ปิด (สีเทา)** = ระบบจะหยุดดึงผลและหยุดส่งหวยนี้ชั่วคราว

> **เคล็ดลับ:** ถ้าหวยตัวไหนมีปัญหา Scraping ไม่ได้ ให้ปิดไว้ก่อน แล้วแจ้งผู้พัฒนา

---

## 4. วิธีดูประวัติส่ง

1. ไปที่เมนู **ประวัติส่ง**
2. จะเห็นรายการส่งผลหวยทั้งหมด เรียงจากล่าสุด
3. แต่ละรายการแสดง:
   - วันเวลาที่ส่ง
   - ชื่อหวย
   - ผลที่ส่ง
   - ช่องทาง (LINE / Telegram)
   - สถานะ (สำเร็จ / ไม่สำเร็จ)
4. สามารถ **กรองตามวัน** หรือ **ค้นหาตามชื่อหวย** ได้
5. กดที่รายการเพื่อดูรายละเอียดเพิ่มเติม เช่น ข้อความที่ส่ง และ Error (ถ้ามี)

---

## 5. วิธีตั้งค่า Telegram Bot

### สร้าง Telegram Bot (ถ้ายังไม่มี)

1. เปิด Telegram แล้วค้นหา **@BotFather**
2. พิมพ์ `/newbot`
3. ตั้งชื่อ Bot (เช่น LottoBot Results)
4. ตั้ง Username (เช่น lottobot_results_bot)
5. BotFather จะส่ง **Token** มาให้ — เก็บไว้ให้ดี

### ตั้งค่าใน Dashboard

1. ไปที่เมนู **ตั้งค่า** > **Telegram**
2. วาง **Bot Token** ที่ได้จาก BotFather
3. กด **บันทึก**
4. กด **ทดสอบการเชื่อมต่อ** เพื่อตรวจสอบว่าใช้งานได้

### เพิ่ม Bot เข้ากลุ่ม Telegram

1. สร้างกลุ่ม Telegram หรือเปิดกลุ่มที่ต้องการ
2. เพิ่ม Bot เข้าไปในกลุ่ม
3. ส่งข้อความอะไรก็ได้ในกลุ่ม 1 ข้อความ
4. กลับมาที่ Dashboard แล้วกด **ดึง Chat ID** — ระบบจะหา Chat ID ของกลุ่มให้
5. กด **บันทึก**

## 6. วิธีจัดการกลุ่ม LINE

### เพิ่มกลุ่ม LINE

1. เพิ่ม LINE Bot เป็นเพื่อนในกลุ่ม LINE ที่ต้องการส่งผลหวย
2. เมื่อเพิ่มแล้ว ระบบจะได้รับ Group ID อัตโนมัติ
3. ไปที่ Dashboard เมนู **ตั้งค่า** > **LINE Groups**
4. จะเห็นกลุ่มใหม่ปรากฏในรายการ
5. กดปุ่ม **เปิดใช้งาน** เพื่อเริ่มส่งผลหวยเข้ากลุ่มนั้น

### จัดการกลุ่ม

- **เปิด/ปิด** การส่งผลหวยแต่ละกลุ่มได้ด้วยสวิตช์
- **ตั้งชื่อกลุ่ม** เพื่อให้จำง่ายขึ้น (ชื่อนี้แสดงใน Dashboard เท่านั้น)
- **เลือกหวย** ที่ต้องการส่งในแต่ละกลุ่ม (ไม่จำเป็นต้องส่งทุกหวย)

> **หมายเหตุ:** ถ้าลบ Bot ออกจากกลุ่ม LINE ระบบจะส่งผลหวยไปกลุ่มนั้นไม่ได้

## 7. วิธีตั้งค่า n8n

n8n เป็นตัวกลางเชื่อม Telegram กับ LINE ถ้าระบบตั้งค่าไว้แล้ว ปกติไม่ต้องทำอะไรเพิ่ม

### ตรวจสอบสถานะ n8n

1. ไปที่ Dashboard เมนู **ตั้งค่า** > **n8n**
2. ดูสถานะ:
   - **เชื่อมต่อสำเร็จ (สีเขียว)** = ปกติ
   - **เชื่อมต่อไม่ได้ (สีแดง)** = มีปัญหา ต้องตรวจสอบ

### แก้ไขเมื่อมีปัญหา

1. ตรวจสอบว่า n8n ยังทำงานอยู่หรือไม่ (เข้า URL ของ n8n)
2. เข้าไปที่ n8n Dashboard
3. ตรวจสอบ Workflow ว่าเปิดอยู่ (Active)
4. ดู Execution History ว่ามี Error อะไร
5. หากแก้ไม่ได้ ให้แจ้งผู้พัฒนา (ในช่วงรับประกัน)

> **สำคัญ:** อย่าแก้ไข Workflow ใน n8n เอง ถ้าไม่แน่ใจ เพราะอาจทำให้ระบบส่งผลหวยไม่ได้

## 8. วิธีตั้งค่า Scraping

ระบบ Scraping จะดึงผลหวยจากเว็บต้นทางอัตโนมัติ ปกติไม่ต้องตั้งค่าอะไร แต่หากต้องการปรับ:

### ดูสถานะ Scraping

1. ไปที่ Dashboard เมนู **จัดการหวย**
2. แต่ละหวยจะมีสถานะ Scraping:
   - **ปกติ (สีเขียว)** = ดึงข้อมูลได้ตามปกติ
   - **มีปัญหา (สีแดง)** = ดึงข้อมูลไม่ได้

### เมื่อ Scraping มีปัญหา

สาเหตุที่พบบ่อย:
- เว็บต้นทางเปลี่ยนโครงสร้างหน้าเว็บ
- เว็บต้นทางล่มชั่วคราว
- เว็บต้นทางบล็อกการเข้าถึง

**วิธีแก้เบื้องต้น:**
1. รอ 1-2 ชั่วโมง แล้วดูอีกครั้ง (กรณีเว็บล่มชั่วคราว)
2. ปิดหวยตัวนั้นไว้ก่อน (กดสวิตช์ปิด)
3. แจ้งผู้พัฒนาเพื่อตรวจสอบและอัปเดต Scraping Script

## 9. คำถามที่พบบ่อย (FAQ)

### ถาม: ข้อมูลจะหายไหมถ้าระบบมีปัญหา?

**ตอบ:** ไม่หาย ข้อมูลทั้งหมดเก็บอยู่ใน Supabase (ฐานข้อมูล Cloud) ซึ่งมีระบบ Backup อัตโนมัติ ถึงแม้เว็บจะล่ม ข้อมูลก็ยังอยู่ครบ

---

### ถาม: LINE ไม่ได้รับผลหวย ทำอย่างไร?

**ตอบ:** ตรวจสอบตามลำดับนี้:
1. เข้า Dashboard ดู **ประวัติส่ง** ว่าระบบส่งหรือยัง
2. ถ้าส่งแล้วแต่ไม่ได้รับ — ตรวจสอบว่า Bot ยังอยู่ในกลุ่มหรือไม่
3. ถ้าระบบไม่ได้ส่ง — ตรวจสอบว่าหวยตัวนั้นเปิดอยู่หรือไม่
4. ตรวจสอบว่ากลุ่ม LINE เปิดใช้งานอยู่หรือไม่
5. หากยังแก้ไม่ได้ ให้แจ้งผู้พัฒนา

---

### ถาม: Telegram ส่งไม่ได้ ทำอย่างไร?

**ตอบ:** ตรวจสอบตามลำดับนี้:
1. เข้า Dashboard ดู **ตั้งค่า** > **Telegram** ว่าสถานะเชื่อมต่อปกติหรือไม่
2. ตรวจสอบว่า Bot ยังอยู่ในกลุ่ม Telegram หรือไม่
3. ตรวจสอบสถานะ n8n ว่าทำงานปกติหรือไม่
4. หากยังแก้ไม่ได้ ให้แจ้งผู้พัฒนา

---

### ถาม: อยากเพิ่มหวยใหม่ได้ไหม?

**ตอบ:** ได้ สามารถเพิ่มหวยใหม่ผ่าน Dashboard ได้เลย แต่ต้องมี URL เว็บต้นทางที่มีผลหวย และอาจต้องให้ผู้พัฒนาเขียน Scraping Script เพิ่มเติม (คิดค่าบริการแยกต่างหาก หากเลยช่วงรับประกัน)

---

### ถาม: ค่าใช้จ่ายรายเดือนมีอะไรบ้าง?

**ตอบ:**
- **Supabase** — ฟรี (Free Plan) สำหรับการใช้งานระดับนี้
- **Vercel** — ฟรี (Hobby Plan) สำหรับการใช้งานระดับนี้
- **n8n** — ขึ้นอยู่กับแพลนที่ใช้ (Self-hosted = ฟรี / Cloud = มีค่าใช้จ่าย)
- **LINE Messaging API** — ฟรี (Free Plan มีโควตาข้อความต่อเดือน)

---

### ถาม: เปลี่ยนรหัสผ่าน Dashboard ได้ไหม?

**ตอบ:** ได้ ไปที่ Dashboard เมนู **ตั้งค่า** > **บัญชีผู้ใช้** > **เปลี่ยนรหัสผ่าน**

---

### ถาม: จะติดต่อผู้พัฒนาอย่างไรเมื่อมีปัญหา?

**ตอบ:** ติดต่อผ่าน Fastwork หรือช่องทางที่ตกลงไว้ (ภายในระยะรับประกัน 30 วัน)
